# Python Concurrency: Parallelism, Threading & Asyncio

Web presentation on concurrency and async in Python.

---

## Prerequisites

| Requirement | Details |
|-------------|---------|
| **Python** | **3.8 or newer** (for the local server). [Download](https://www.python.org/downloads/). |
| **Browser** | Any modern browser (Chrome, Firefox, Edge, Safari) with JavaScript enabled. |
| **Internet** | Required for the interactive “▶ Run” exercises (they use Pyodide in the browser). |

No Node.js, npm, or other packages are required. The presentation runs with Python’s built-in HTTP server only.

---

## How to run

1. Install **Python** if you haven't.
2. Start the server:
   ```text
   python -m http.server 8888
   ```
3. Open your browser and go to: **http://localhost:8888**
4. To stop the server, press **Ctrl+C** in the terminal.
